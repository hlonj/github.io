## 앞에 `샵샵` 을 붙이면 글씨가 커지지롱

글씨 사이에 *별별* 을 붙여주면 글씨가 기울어집니다 `{ .. }` 괄호는 뭔지 모름`let`:

{% highlight js %}
var a = 2;

{
    let a = 3;
    console.log( a );   // 3
}

console.log( a );       // 2
{% endhighlight %}
